＜jQueryの使用方法＞

※jquery-X.X.X.min.jsのminは「minified」の略で「圧縮版」を意味しています。
jQueryを使用する場合には、圧縮版である jquery-X.X.X.min.js を使用します。


jQuery…つまり、jquery-X.X.X.min.js をどうやって使うかですが、
HTMLの<head>〜</head>内で使用します。
というか記述します。

----------------------------------------------------------------------------------------------------------
例)
<head>

<script type="text/javascript" src="js/jquery-X.X.X.min.js"></script>

</head>

----------------------------------------------------------------------------------------------------------
